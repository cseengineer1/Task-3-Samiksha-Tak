# SQL Data Analysis Project

## Overview

This project was completed as part of the DecodeLabs Data Analytics Internship. The objective of the project was to analyze a sales dataset using SQL and extract useful insights from the data.

## Dataset

The dataset contains sales transaction records including product details, quantities, prices, payment methods, order status, referral sources, and total order values.

## Tools Used

* SQLite Online
* Microsoft Excel
* SQL

## SQL Operations Performed

* SELECT
* WHERE
* ORDER BY
* GROUP BY
* COUNT()
* SUM()
* AVG()
* MIN()
* MAX()

## Project Work

During this project, I performed data exploration and analysis using SQL queries. Different SQL operations were used to filter records, sort data, calculate revenue, count transactions, find average order values, and analyze product performance.

The project also included:

* Product-wise revenue analysis
* Payment method analysis
* Order status analysis
* Top-performing product identification
* Revenue-based reporting

## Key Learning Outcomes

Through this project, I gained practical experience in:

* Writing SQL queries
* Data filtering and sorting
* Data aggregation techniques
* Business data analysis
* Extracting insights from structured datasets

## Conclusion

This project helped me strengthen my understanding of SQL and its application in data analytics. It provided hands-on experience in working with real-world data and generating meaningful insights through structured queries.
